# mPanel

**mPanel By Melbourne Roleplay** is an advanced management and operations platform designed specifically for ER:LC communities and large-scale roleplay environments.

Unlike traditional admin systems or CAD platforms, mPanel functions as a **unified operations hub**, combining moderation, staffing, communications, incident tracking, automation, analytics, and server management into a single cohesive platform.

Built with scalability, performance, and usability in mind, MELCORE streamlines community operations while providing staff teams with powerful tools to manage day-to-day activities efficiently.


## Installation

```bash
git clone https://github.com/itx-w4sley/Melbourne-Roleplay-Website-V1
cd melcore
composer install
```
